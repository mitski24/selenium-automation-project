Project Overview-

This project automates end-to-end user workflows on the SauceDemo website using Selenium WebDriver with Python.
It validates critical functionalities such as login, adding/removing products, cart validation, checkout process, and logout.
The automation framework follows the Page Object Model (POM) design pattern to ensure code readability, reusability, and maintainability.

Tech Stack
Python 3.x
Selenium WebDriver
PyTest
WebDriver Manager
Google Chrome Browser

Steps to Execute the Scripts-

1.Clone or Download the Project

git clone <your-github-repo-link>

or download ZIP and extract.

2.Navigate to Project Folder

cd saucedemo_automation

3.Install Required Dependencies

pip install -r requirements.txt

4.Run the Automation Tests

pytest -v
